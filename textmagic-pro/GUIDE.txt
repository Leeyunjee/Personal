3️⃣ 수익화 설정

[STEP 1] OpenAI API 키 설정 (AI 기능 활성화)

1. https://platform.openai.com/signup 에서 계정 생성
2. https://platform.openai.com/api-keys 에서 API 키 발급
3. .env.local 파일 열기
4. OPENAI_API_KEY=여기에_API_키_붙여넣기
5. 서버 재시작

💡 비용: GPT-4o-mini 사용시 1000회 요청당 약 $0.15
   → Pro 고객 1명 = $9 수익, 비용 약 $0.08

---

[STEP 2] Stripe 결제 연동 (실제 결제 받기)

1. https://dashboard.stripe.com 에서 계정 생성

2. API 키 발급:
   - 개발자 → API 키
   - Secret Key 복사 → .env.local의 STRIPE_SECRET_KEY에 붙여넣기
   - Publishable Key 복사 → STRIPE_PUBLISHABLE_KEY에 붙여넣기

3. 제품 생성:
   - 제품 → 제품 추가
   - Pro: $9/월 반복 결제
   - Business: $29/월 반복 결제

4. Price ID 복사:
   - 각 제품의 가격 ID를 .env.local에 추가
   - STRIPE_PRO_PRICE_ID=price_xxxxx
   - STRIPE_BUSINESS_PRICE_ID=price_xxxxx

5. 웹훅 설정 (배포 후):
   - 개발자 → 웹훅
   - 엔드포인트 추가: https://yoursite.com/api/stripe/webhook
   - 이벤트: checkout.session.completed, customer.subscription.deleted
   - Webhook Secret → STRIPE_WEBHOOK_SECRET에 추가

4️⃣ 웹 배포 (Vercel - 무료)

[배포 방법]

1. GitHub에 코드 업로드:
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/your-username/textmagic-pro.git
   git push -u origin main

2. Vercel 배포:
   - https://vercel.com 접속
   - GitHub 계정으로 로그인
   - "New Project" 클릭
   - textmagic-pro 저장소 선택
   - "Deploy" 클릭

3. 환경 변수 설정:
   - Vercel 프로젝트 → Settings → Environment Variables
   - .env.local의 모든 변수 추가

4. 커스텀 도메인 (선택):
   - Settings → Domains
   - 원하는 도메인 추가 (예: textmagic.pro)

💡 Vercel 무료 플랜으로 충분히 시작 가능!

6️⃣ FAQ

Q: OpenAI API 없이도 작동하나요?
A: 네, 데모 모드로 작동합니다. 실제 AI 응답을 받으려면 API 키가 필요합니다.

Q: Stripe 없이도 사용 가능한가요?
A: 네, 무료 서비스로만 운영 가능합니다. 수익화하려면 Stripe 설정이 필요합니다.

Q: 서버 비용이 드나요?
A: Vercel 무료 플랜으로 시작 가능. 트래픽 증가시 월 $20 정도.

Q: OpenAI 비용은 얼마인가요?
A: GPT-4o-mini 기준, 1000회 요청당 약 $0.15. 매우 저렴합니다.

Q: 데이터는 어디에 저장되나요?
A: 로컬에서는 SQLite(data/textmagic.db), 배포 시 별도 DB 설정 필요.

Q: 코드를 수정할 수 있나요?
A: 물론입니다! src/ 폴더의 모든 파일을 자유롭게 수정하세요.
